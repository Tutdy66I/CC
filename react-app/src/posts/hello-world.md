This is the first post on this blog. It demonstrates how the blog system works.

## Features

- **Markdown rendering** via `react-markdown`
- **Frontmatter** using HTML comments for title, date, and tags
- **View switching** with no router — just `useState`
- **Tag filtering** — click a tag to filter posts

  return `Hello, ${name}!`
